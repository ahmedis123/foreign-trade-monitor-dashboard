
# foreign-trade-monitor-dashboard
# مرصد التجارة الخارجية السودانية

مرصد التجارة الخارجية السودانية هو لوحة تحكم تفاعلية تعرض البيانات الاقتصادية المتعلقة بالتجارة الخارجية للسودان. تم تصميم هذه اللوحة باستخدام تقنيات الويب الحديثة مثل **Tailwind CSS** و **Chart.js**، وهي تهدف إلى توفير تحليل دقيق لأداء التجارة الخارجية السودانية خلال الفترات الزمنية المختلفة.

## محتويات المشروع

يتضمن هذا المشروع لوحة تحكم تعرض الإحصائيات والمخططات التالية:
- **إجمالي الصادرات**: يعرض إجمالي قيمة الصادرات السودانية للفترة من يناير إلى يونيو 2025.
- **إجمالي الواردات**: يعرض إجمالي قيمة الواردات السودانية للفترة نفسها.
- **الميزان التجاري**: يظهر الفرق بين الصادرات والواردات ويعرض تغيراته مقارنة بالعام السابق.
- **مخططات تفاعلية**: بما في ذلك مخططات تاريخية، هيكل الصادرات، الشركاء التجاريين الرئيسيين، والأداء الشهري.
- **جدول البيانات**: جدول يوضح الميزان التجاري للسودان خلال الفترة من 2015 إلى 2025.

## التقنيات المستخدمة

- **HTML5**: هيكل الصفحة.
- **CSS3** (مع **Tailwind CSS**): تصميم واجهة المستخدم بشكل مرن ومتجاوب.
- **JavaScript**: للتفاعلات والديناميكية في الصفحة.
- **Chart.js**: لإنشاء الرسوم البيانية التفاعلية.
- **GitHub Pages**: لنشر المشروع عبر الإنترنت.
